PMDG 737-800 HOTKEY REFERENCE

Application Navigation:
  Ctrl+1     Focus Sections List
  Ctrl+2     Focus Panels List
  Ctrl+3     Focus Status Display field of the current panel (refreshes it if empty)

Tip: Press the same bracket key again to cancel a mode you activated by accident.

OUTPUT MODE - Press right bracket to activate

MCP Readouts:
  Shift+H    Read MCP Heading (includes HDG SEL/LNAV status)
  Shift+S    Read MCP Speed (includes LVL CHG status, shows blank if FMC-controlled)
  Shift+A    Read MCP Altitude (includes VNAV/LVL CHG/ALT HOLD status)
  Shift+V    Read MCP Vertical Speed (includes VS engaged status)

Altitude:
  Q          Read Altitude AGL
  A          Read Altitude MSL

Airspeed:
  S          Read IAS (Indicated Airspeed)
  T          Read TAS (True Airspeed)
  G          Read Ground Speed
  M          Read Mach Number

Heading:
  H          Read Magnetic Heading
  U          Read True Heading

Navigation:
  N          Read NAV 1 and NAV 2 radio info (frequency, ident, localizer, glideslope, DME)

Fuel:
  F          Read Fuel Quantity per tank (pounds)
  Shift+F    Read Fuel Quantity per tank (kilograms)

Weight:
  W          Read Gross Weight (pounds)
  Shift+W    Read Gross Weight (kilograms)

Flaps:
  L          Read Flap Position

Landing Gear:
  Shift+G    Read Gear Position

Altimeter:
  B          Read altimeter setting (announces in hPa and inHg;
             "Altimeter standard" if at 29.92 inHg)

Vertical:
  V          Read Vertical Speed
  Ctrl+Shift+R  Read last landing rate (touchdown vertical speed, feet per minute)
  Ctrl+Shift+G  Read last landing peak g-force

Weather:
  I          Wind Info
  O          Read Outside Temperature (Celsius)

Time:
  Z          Read Local Time at aircraft position. Spoken as
             "HH:MM <time-zone name>" (e.g. "16:38 Eastern Daylight Time"
             near KJFK), or "HH:MM:SS <time-zone name>" (e.g.
             "20:30:45 British Summer Time") if "Include seconds in
             time announcements" is ticked in Announcement Settings.
             The time-zone name is looked up from the aircraft's
             geographic position and is DST-aware.
  Shift+Z    Read Zulu / UTC Time. Spoken as "HH:MMZ" (e.g. "03:30Z"),
             or "HH:MM:SSZ" (e.g. "00:15:30Z") when seconds are on.

Transponder:
  X          Read Squawk Code

Distance:
  D          Read Distance to Destination (with FMC Settings > Enhanced
             distance: speaks PROG-page distance + ETA in Z time + landing fuel)
  Shift+D    Read Distance to Top of Descent (Enhanced mode: TOC if not yet
             passed, then step climb if not "NONE", then TOD — all with PROG
             ETA in Z time)

Location:
  C          Read Nearest City

Trim:
  Shift+T    Toggle Trim Announcements On/Off

Tools:
  Ctrl+H     Activate/deactivate Hand Fly mode
  Ctrl+F     Activate/deactivate Waypoint Flight Director (synthetic audio FD:
             hand-fly to the tracked waypoint slots in order; stereo pan = bank
             command, tone pitch = climb/descend). Output mode; no Hand Fly
             needed. Set crossing altitudes in the Track Fix window (Shift+F).
  Ctrl+V     Activate/deactivate Visual Guidance mode (requires Hand Fly mode)
  Ctrl+T     Activate/deactivate Takeoff Assistance
  Ctrl+D     Read Distance to Selected Runway
  Ctrl+I     Read ILS Guidance System Report
  Ctrl+0     Read Approach Capability

Taxi Guidance (turn-by-turn taxi assistance with audio steering tone):
  Y          Taxi Status (current taxiway, next turn, distance to destination)
  Ctrl+Y     Repeat Current Taxi Instruction
  Alt+Y      Where Am I (announces current taxiway, gate, or runway at nearest airport)
  (Open the form with Input mode > Shift+Y)

Waypoint Tracking (use Fix Track window to add):
  1          Track Slot 1
  2          Track Slot 2
  3          Track Slot 3
  4          Track Slot 4
  5          Track Slot 5

AI Display Reading (requires Gemini API key in File > Gemini Settings):
  Alt+P      Read PFD (use Ctrl+2 cockpit view first)
  Alt+N      Read Navigation Display (use Ctrl+2 cockpit view first)
  Alt+I      Read ISFD backup instruments (use Ctrl+2 cockpit view first)
  Alt+E      Read EICAS display (use Ctrl+2 cockpit view first)
  Alt+D      Describe outside scene

Traffic and Weather:
  R          Announce Tracked TCAS Traffic
  Ctrl+R     Open TCAS Traffic Window
  Shift+R    Open Weather Radar Window (auto-detects ActiveSky and silently
             uses richer data — surface wind+gust, ceiling, QNH, turbulence,
             METAR-decoded precipitation — when running. Falls back to MSFS
             SimVars when AS isn't running. Visibility shown in both km and
             statute miles either way.)
  Alt+G      Announce nearest ground traffic (up to 3 aircraft on ground with
             direction and distance; auto-alerts also fire at 500/250/100 ft
             zone transitions while taxiing)

Displays and Windows:
  Ctrl+M     PMDG Monitor Manager
  Shift+E    Route viewer, procedure planner and airport lookup
  Shift+L    Location Info
  Shift+M    Show METAR Report (VATSIM, plus ActiveSky METAR if AS detected)
  Shift+B    SimBrief Briefing
  Shift+C    Show Checklist

INPUT MODE - PMDG EFB Tablet:
  Shift+T    PMDG EFB Tablet - Captain tablet
  Ctrl+Shift+T  PMDG EFB Tablet - First Officer tablet

OUTPUT MODE - Cold Temperature Altitude Correction:
  Ctrl+Shift+T  Cold Temperature Altitude Correction (offline calculator: field elevation + temperature + published altitudes -> corrected altitudes)

Hand Fly Mode Hotkeys:
  A          Read MSL
  S          Read IAS
  D          Read Distance to Destination Runway
  H          Read Magnetic Heading
  P          Read Aircraft Pitch
  B          Read Aircraft Bank
  Q          Read AGL
  V          Read VS

Visual Guidance Mode Hotkeys:
  F          Read Target VS

INPUT MODE - Press [ to activate

Teleport:
  Shift+R    Runway Teleport
  Shift+G    Gate Teleport

Flight Planning:
  Shift+F    Fix/Waypoint Tracking Window
  Shift+D    Select Destination Runway

MCP Set Values:
  Ctrl+H     Set Heading (opens dialog with Intervene, HDG SEL, LNAV toggles)
  Ctrl+S     Set Speed (opens dialog with Intervene, IAS/Mach Mode toggles)
  Ctrl+A     Set Altitude (opens dialog with Intervene, VNAV, Level Change, ALT HOLD toggles)
  Ctrl+V     Set Vertical Speed (opens dialog with VS Engage toggle)

Altimeter:
  Ctrl+B     Set altimeter (input dialog accepts hPa 900-1060 or
             inHg 26.50-31.30; branches on entered magnitude, so both
             "1013" and "29.92" work, locale-safe; includes a Standard
             button that sets the altimeter to STD / 29.92)

Radios:
  Ctrl+N     Set NAV radios (input dialog with NAV 1 and NAV 2 frequency
             and course; pre-filled with current values, applies all on
             Set; frequency 108.00-117.95 MHz, course 0-359 degrees)

Taxi Guidance:
  Shift+Y    Open Taxi Assist Form (enter ATC taxi clearance, start guidance)
  Y          Continue past current hold-short point
  Ctrl+Y     Stop Taxi Guidance
  Shift+X    Open Landing Exit Planner (pick a runway exit taxiway before
             touchdown; guidance auto-starts when you land — reuses the ILS
             destination runway if one is set; announces headwind/tailwind on
             runway selection so you can pick an exit that matches the rollout)

General:
  Escape     Exit active hotkey mode

CDU Window (open with [ then Shift+M):

Navigation:
  Up/Down        Navigate display lines
  PageUp         Previous CDU page
  PageDown       Next CDU page
  Alt+Up         Previous CDU page (alternative)
  Alt+Down       Next CDU page (alternative)
  Backspace      CLR (if scratchpad has text) / DEL (if scratchpad empty)

Line Select Keys:
  Ctrl+1-6       Left LSK 1-6
  Alt+1-6        Right LSK 1-6

Text Input:
  Alt+S          Focus scratchpad input field
  Enter          Submit typed text to CDU scratchpad
  Alt+Home       Return focus to display output

CDU Page Buttons:
  Alt+I          Init Ref
  Alt+R          RTE (Route)
  Alt+D          Dep/Arr (Departure/Arrival)
  Alt+V          VNAV
  Alt+F          Fix
  Alt+G          Legs
  Alt+H          Hold
  Alt+P          Prog (Progress)
  Alt+M          Menu
  Alt+B          CLB (Climb)
  Alt+Z          CRZ (Cruise)
  Alt+X          DES (Descent)
  Alt+N          N1 Limit
  Alt+E          Execute
  Alt+C          CLR/DEL (Clear or Delete)

Note: in FMC Settings (File > FMC Settings) you can switch to alternate
line-select keys: F1-F6 for L1-L6 and F7-F12 for R1-R6. With this on,
the default Ctrl+1-6 / Alt+1-6 chords are disabled.

Automatic Announcements:
  - MCP value changes (speed, heading, altitude, vertical speed)
  - Autopilot mode changes (AP, AT, LNAV, VNAV, LVL CHG, HDG SEL, VS, ALT HOLD, VOR/LOC, APP)
  - Master Warning / Master Caution (with cleared state)
  - All annunciator lights (on and off)
  - Speed brake position changes
  - Flap position changes
  - FMC V-speeds (V1, VR, V2, Cruise Altitude)
  - APU status (running / shut down)
  - IRS alignment
  - Elevator trim changes (toggle with Shift+T)
  - Altimeter setting changes
  - Thousand-foot altitude crossings
  - COM frequency changes
  - Squawk code changes
  - CDU page changes and scratchpad updates
