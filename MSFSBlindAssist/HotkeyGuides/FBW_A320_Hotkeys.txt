FBW A320 HOTKEY REFERENCE

Application Navigation:
  Ctrl+1     Focus Sections List
  Ctrl+2     Focus Panels List

Tip: Press the same bracket key again to cancel a mode you activated by accident.


OUTPUT MODE - Press right bracket to activate

AI features:
Alt+d  Describe Scene

FCU Controls:
  Shift+H    Read FCU Heading
  Shift+S    Read FCU Speed
  Shift+A    Read FCU Altitude
  Shift+V    Read FCU VS/FPA

Altitude:
  Q          Read Altitude AGL
  A          Read Altitude MSL

Airspeed:
  S          Read IAS (Indicated Airspeed)
  T          Read TAS (True Airspeed)
  G          Read Ground Speed
  M Read Mach Number.

Heading:
  H          Read Magnetic Heading
  U          Read True Heading

Vertical:
  V          Read Vertical Speed

Weather:
  I          Wind Info
  O          Read Outside Temperature (Celsius)

Time:
  Z          Read Local Time at aircraft position. Spoken as
             "HH:MM <time-zone name>" (e.g. "16:38 Eastern Daylight Time"
             near KJFK), or "HH:MM:SS <time-zone name>" (e.g.
             "20:30:45 British Summer Time") if "Include seconds in
             time announcements" is ticked in Announcement Settings.
             The time-zone name is looked up from the aircraft's
             geographic position and is DST-aware.
  Shift+Z    Read Zulu / UTC Time. Spoken as "HH:MMZ" (e.g. "03:30Z"),
             or "HH:MM:SSZ" (e.g. "00:15:30Z") when seconds are on.

Transponder:
  X          Read Squawk Code

Location:
  C          Read Nearest City

Flight planning:
  W Next waypoint information

Fuel:
  F          Read Total Fuel Quantity

Tools:
CTRL+h, Activate/deactivate Hand fly mode, Note, enables more hotkeys, advanced feature.
CTRL+v, activate/deactivate visual guidance mode, note, does not require hand fly mode. Enables the same single-letter quick-access hotkeys hand fly mode does, plus F for target VS.
CTRL+T, Takeoff Assistance
CTRL+D, Distance to Selected Runway
CTRL+I, ILS guidance system
CTRL+E, Enable Engine and warning display auto announcements

Taxi Guidance (turn-by-turn taxi assistance with audio steering tone):
  Y          Taxi Status (current taxiway, next turn, distance to destination)
  Ctrl+Y     Repeat Current Taxi Instruction
  Alt+Y      Where Am I (announces current taxiway, gate, or runway at nearest airport)
  (Open the form with Input mode > Shift+Y)

Traffic and Weather:
  R          Announce Tracked TCAS Traffic
  Ctrl+R     Open TCAS Traffic Window
  Shift+R    Open Weather Radar Window (auto-detects ActiveSky and silently
             uses richer data — surface wind+gust, ceiling, QNH, turbulence,
             METAR-decoded precipitation — when running. Falls back to MSFS
             SimVars when AS isn't running. Visibility shown in both km and
             statute miles either way.)
  Alt+G      Announce nearest ground traffic (up to 3 aircraft on ground with
             direction and distance; auto-alerts also fire at 500/250/100 ft
             zone transitions while taxiing)

Displays and windows:
shift+e, Route viewer, procedure planning and airport lookup
  Shift+F, Fuel and payload window
  Shift+T,    Toggle Trim Announcements
  Shift+U,    Engine and warning display window
  Shift+Y,    SD Status window
  Shift+N,    Navigation display
  Shift+L,    Location Info
  Shift+M,    Show METAR Report (VATSIM, plus ActiveSky METAR if AS detected)
  Shift+B,    SimBrief Briefing
  Shift+C,    Show Checklist
  Shift+P,    Show PFD Window

Speed Tape:
  Shift+1    Read O Speed (GD)
  Shift+2    Read S-Speed
  Shift+3    Read F-Speed
  Shift+4    Read VLS (Minimum Selectable Speed)
  Shift+5    Read VS (Stall Speed)
  Shift+6    Read VFE Speed

Waypoint tracking slots:
1, track slot 1
2, track slot 2
3, track slot 3
4, track slot 4
5, track slot 5

Hand fly mode hotkeys:
A, read MSL
S, Read IAS
D, Read distance to destination runway
H, Read MH
P, Read aircraft's pitch
B, Read aircraft's bank
Q, read AGL
V, Read VS

Visual guidance mode hotkeys (same single-letter set as hand fly mode, plus F):
A, read MSL
S, Read IAS
D, Read distance to destination runway
H, Read MH
P, Read aircraft's pitch
B, Read aircraft's bank
Q, read AGL
V, Read VS
F, Read target VS

Note: both modes share the quick-access hotkeys. Activate hand fly mode, visual guidance mode, or both — the single-letter keys H, V, Q, S, D, B, P, A work in either. F is visual-guidance-specific (announces "visual guidance not active" if pressed when VG is off).

Ground Services:
  Ctrl+G     Read latest GSX tooltip (no need to open the Access GSX window)
             In the Access GSX window, press F5 to open the GSX menu, 1-9/0/A-E
             to choose menu options, C to open GSX Settings, and Escape to hide
             the window. If multiple GSX services are active, use the Active
             services combo to choose which service drives tooltip announcements.

SayIntentions:
  Ctrl+S        Read last SayIntentions transmission
  Ctrl+Shift+S  Read SayIntentions assigned gate and runway
  Input mode Alt+Shift+S  Build taxi route from SayIntentions clearance or gate

INPUT MODE - Press left bracket to activate

Teleport:
  Shift+R    Runway Teleport
  Shift+G    Gate Teleport

Flight planning:
shift+f, Fix/waypoint tracking
  Shift+D    Select Destination Runway

Autopilot:
  Shift+A    Toggle Autopilot 1
  Ctrl+O     Toggle Autopilot 2
  Shift+P    Toggle Approach Mode

FCU Push/Pull:
  Shift+1    Push Heading Knob
  Ctrl+1     Pull Heading Knob
  Shift+2    Push Altitude Knob
  Ctrl+2     Pull Altitude Knob
  Shift+3    Push Speed Knob
  Ctrl+3     Pull Speed Knob
  Shift+4    Push VS Knob
  Ctrl+4     Pull VS Knob

FCU Set Values:
  Ctrl+H     Set Heading Value
  Ctrl+S     Set Speed Value
  Ctrl+A     Set Altitude Value
  Ctrl+V     Set VS Value

Taxi Guidance:
  Shift+Y    Open Taxi Assist Form (enter ATC taxi clearance, start guidance)
  Y          Continue past current hold-short point
  Ctrl+Y     Stop Taxi Guidance
  Shift+X    Open Landing Exit Planner (pick a runway exit taxiway before
             touchdown; guidance auto-starts when you land — reuses the ILS
             destination runway if one is set; announces headwind/tailwind on
             runway selection so you can pick an exit that matches the rollout)

Ground Services:
  Alt+G      Open Access GSX window (accessible GSX Ground Services Pro menu)


General:
  Escape     Exit active hotkey mode
