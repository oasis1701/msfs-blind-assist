FLYBYWIRE A380X — HOTKEY REFERENCE

Quick key list only. For what each feature does and step-by-step how to use it
(the MCDU, flyPad, checklists, displays, flight planning, systems, a full-flight
walkthrough), see the full manual: docs/a380-manual.html.

Modes: press ] (right bracket) for OUTPUT mode, [ (left bracket) for INPUT mode,
Escape to exit. A few chords are direct (no mode). Press the same bracket again
to cancel a mode activated by accident.


DIRECT (no mode)
  Ctrl+1     Focus Sections list
  Ctrl+2     Focus Panels list
  Ctrl+3     Focus Status Display field of the current panel (refreshes it if empty)
  Ctrl+M     Monitor Manager (choose what auto-announces)
  Ctrl+E     Toggle ECAM E/WD call-outs
  Ctrl+Shift+B  Airport Map and BTV (OANS) window — see "INSIDE THE OANS WINDOW" below

  Shift+A    FCU Altitude
  Shift+V    FCU Vertical Speed / FPA

  Flight data
  Q          Altitude AGL

  T          TAS
  G          Ground Speed

  H          Magnetic Heading
  U          True Heading

  Shift+D    Distance + time to Top of Descent, e.g. "145 miles to top of
             descent: 00:16:58" (once computed in flight)
             (Q / A / Shift+A read in metres when the FCU MTRS button is on)
  Ctrl+W     ND active (TO) waypoint: name, distance, and bearing
  Ctrl+Shift+R  Last landing rate (touchdown vertical speed, feet per minute)
  Ctrl+Shift+G  Last landing peak g-force

  Characteristic speeds
  Shift+1    Green Dot speed
  Shift+2    S speed (slat retraction)
  Shift+3    F speed (flap retraction)
  Shift+4    VLS (minimum selectable)
  Shift+5    VS (stall)

  Weather / time / misc

  O          Outside Temperature (Celsius)
  Z          Local Time
  Shift+Z    Zulu / UTC Time
  X          Squawk Code
  C          Nearest City
  Shift+T    Toggle trim announcements

  Traffic / weather windows
  R          Announce TCAS traffic
  Ctrl+R     Show TCAS window
  Shift+W    Weather Radar
  Shift+M    METAR report
  Ctrl+Shift+T  Cold Temperature Altitude Correction (offline calculator: field elevation + temperature + published altitudes -> corrected altitudes)

  Display windows
  Alt+E      E/WD (Engine/Warning Display) pop-out window — the whole E/WD (engine
             parameters + the live ECAM memo / warning lines). Read with the arrow
             keys; it auto-refreshes, F5 refreshes now, Escape closes.
  Alt+I      Standby Instruments (ISIS) window
  Alt+D      Describe Scene (AI)

  Checklists
  Shift+C    Static text checklist (cold-and-dark to shutdown)
  Ctrl+Shift+C  Live Electronic Checklist (ECL) window -- the real cockpit
                checklists + active ECAM procedures (see below)

  Taxi

  Alt+Y      Where am I (ground)

  Tools
  Ctrl+H     Hand-fly mode
  Ctrl+V     Visual guidance (needs hand-fly)
  Ctrl+T     Takeoff Assistance
  Ctrl+D     Distance to selected runway
  Ctrl+I     ILS guidance
  Ctrl+G     Read latest GSX tooltip

INSIDE THE MCDU / MFD WINDOW
  Up / Down  Move through lines
  Enter      Activate the line (send scratchpad to field; open a combo box)
  (type)     Type into the scratchpad, then Enter
  Backspace  CLR (scratchpad first, then a displayed info/error message, then field)
  Alt+C      CLR (same as Backspace)
  Delete     Clear the selected field
  Ctrl+Up    Previous page (PageUp also)
  Ctrl+Down  Next page (PageDown also; scrolls the F-PLN window)
  F5         Refresh

  Page buttons (Alt+letter)
  Alt+I  INIT     Alt+F  F-PLN    Alt+P  PERF      Alt+L  FUEL & LOAD
  Alt+R  RAD NAV  Alt+S  SEC FPL  Alt+D  DIR
  Alt+A  ATC COM  Alt+V  SURV     Alt+C  CLR       Alt+H  Refresh
  Alt+N  Toggle weight units (kilograms / pounds) for MSFSBA read-outs
  (No UP/DOWN buttons — page stepping is Ctrl+Up/Down or PageUp/PageDown.)
  The "go to MFD page" combo reaches ALL pages; arrow to one and it navigates
  once your selection settles.


INSIDE THE RADIO MANAGEMENT PANEL (RMP) WINDOW (Ctrl+Shift+R opens it)
  The RMP screen is read-only text — read it with the arrow keys. It refreshes
  automatically and announces any RMP message (e.g. FREQ NOT VALID) as it appears.
  Pick a page, then type the digits right on the screen (no separate entry box —
  the screen is read-only, so the digits go to the radio, not the text):
  Alt+V            VHF page          Alt+T   Transponder (SQWK) page
                   (HF / TEL / NAV / MENU are not modelled by FBW — those do nothing)
  Alt+Home         Jump back to the screen (e.g. from a page button)

  VHF page (left keys on Ctrl, right keys on Alt — same scheme as the MCDU):
  Ctrl+1 / 2 / 3   Select radio 1 / 2 / 3   (or F1/2/3 with alternate MCDU keys on)
  Alt+1 / 2 / 3    Swap that radio's active and standby   (or F7/8/9)
                   Type the frequency (keyed live, no decimal — it auto-completes:
                   118500 or 1185 both give 118.500). Enter loads it into standby and
                   reads it back, e.g. "VHF standby 1, 118.500"; Backspace deletes a
                   digit; Alt+C fully clears a stuck entry. Then Alt+1/2/3 swaps it
                   active (the pilot's call).

  Transponder (SQWK) page:
  Type the 4 squawk digits (0 to 7). It sets automatically on the 4th digit (or press
  Enter) and reads back "Squawk 2222".   Alt+I   sends IDENT.

  Side combo (Captain / First Officer / Overhead) switches which RMP you control —
  the screen, the announcements and the keys all follow it.


INSIDE THE LIVE CHECKLIST (ECL) WINDOW (Ctrl+Shift+C opens it)
  The checklist is shown automatically when the window opens and hidden when it
  closes -- no need to toggle it.
  Up / Down  Move the checklist cursor and read each line
  Enter      Open the highlighted checklist, or tick the highlighted item
  Backspace  Step back to the menu
  F5         Refresh
  (the Abnormal procedures button lists active ECAM procedures)


INSIDE THE OANS WINDOW (Ctrl+Shift+B opens it) -- airport map + Brake-To-Vacate (BTV)
  The OANS airport map only draws at the closest ND range, so opening the window
  AUTO-ZOOMS the Captain ND to the airport map for you (you don't see the ND, so this
  is harmless). Give it a second; press F5 if it still says it's loading. Read the
  controls with the arrow keys; Enter activates a button / opens a list; type into a
  field then Enter; Escape closes. It's a tool for the GROUND and the terminal area
  (taxi-out and approach) -- away from an airport it will say it's not available.
  Three tabs across the top -- arrow to one and press Enter:
   ARPT SEL  Pick the airport: choose the entry mode (ICAO / IATA / CITY NAME), type
             the code into the field + Enter, then press DISPLAY AIRPORT. ORIGIN /
             DEST / ALTN jump to your flight-plan airports.
   MAP DATA  Brake-To-Vacate: pick the landing RWY, read RUNWAY LENGTH and the BTV
             STOP DISTANCE, choose the exit (TWY / STAND / OTHER). BTV MANUAL CONTROL
             is the fallback. (The dry/wet stop distances are also exposed as readouts
             in the panels and announced during the landing rollout.)
   STATUS    Active / second airport database + SWAP, and the database identifier.

Thrust levers auto-announce the detent reached (Idle / Climb / Flex MCT / TOGA /
Reverse). Most cockpit controls are in the Sections/Panels lists (Ctrl+1/Ctrl+2).
Full explanations: docs/a380-manual.html.
