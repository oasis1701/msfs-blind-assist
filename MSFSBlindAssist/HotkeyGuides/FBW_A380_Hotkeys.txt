FLYBYWIRE A380X HOTKEY REFERENCE

Application Navigation:
  Ctrl+1     Focus Sections List
  Ctrl+2     Focus Panels List
  Ctrl+3     Focus Status Display field of the current panel (refreshes it if empty)

Tip: Press the same bracket key again to cancel a mode you activated by accident.
For step-by-step explanations of every feature (MCDU, flyPad, checklists, displays,
flight planning, systems, a full-flight walkthrough), see Guides/a380-manual.html.

OUTPUT MODE - Press right bracket to activate

FCU Controls:
  Shift+H    Read FCU Heading
  Shift+S    Read FCU Speed (knots or Mach)
  Shift+A    Read FCU Altitude
  Shift+V    Read FCU V/S or FPA

Altitude:
  Q          Read Altitude AGL
  A          Read Altitude MSL
  (Q and A read in metres when the FCU MTRS button is on)

Airspeed:
  S          Read IAS (Indicated Airspeed)
  T          Read TAS (True Airspeed)
  G          Read Ground Speed
  M          Read Mach Number

Heading:
  H          Read Magnetic Heading
  U          Read True Heading

Navigation:
  N          Read NAV 1 and NAV 2 radio info (frequency, ident, localizer, glideslope, DME)
  Ctrl+W     Read ND active (TO) waypoint: name, distance, bearing

Characteristic Speeds:
  Shift+1    Read Green Dot Speed
  Shift+2    Read S Speed (slat retraction)
  Shift+3    Read F Speed (flap retraction)
  Shift+4    Read VLS (minimum selectable speed)
  Shift+5    Read VS (stall speed)
  Shift+6    Read VFE Next (next flap limit)

Fuel:
  F          Read Fuel On Board (pounds)
  Shift+F    Read Fuel On Board (kilograms)
  Ctrl+1-8   Read one fuel tank in pounds (Alt+1-8 for kilograms):
             1 Feed 1, 2 Feed 2, 3 Feed 3, 4 Feed 4,
             5 Outer tanks (left and right), 6 Mid tanks (left and right),
             7 Inner tanks (left and right), 8 Trim tank

Weight:
  W          Read Gross Weight (pounds) + center of gravity
  Shift+W    Read Gross Weight (kilograms) + center of gravity

Flaps:
  L          Read Flap Position

Landing Gear:
  Shift+G    Read Gear Position

Altimeter:
  B          Read Altimeter Setting (hPa and inHg, or Standard)

Vertical:
  V          Read Vertical Speed
  Ctrl+Shift+R  Read last landing rate (touchdown vertical speed, feet per minute)
  Ctrl+Shift+G  Read last landing peak g-force

Approach:
  Ctrl+0     Read Approach Capability (CAT rating)

Weather:
  I          Wind Info
  O          Read Outside Temperature (Celsius)

Time:
  Z          Read Local Time at aircraft position. Spoken as
             "HH:MM <time-zone name>" (e.g. "16:38 Eastern Daylight Time"
             near KJFK), or "HH:MM:SS <time-zone name>" (e.g.
             "20:30:45 British Summer Time") if "Include seconds in
             time announcements" is ticked in Announcement Settings.
             The time-zone name is looked up from the aircraft's
             geographic position and is DST-aware.
  Shift+Z    Read Zulu / UTC Time. Spoken as "HH:MMZ" (e.g. "03:30Z"),
             or "HH:MM:SSZ" (e.g. "00:15:30Z") when seconds are on.

Transponder:
  X          Read Squawk Code

Distance:
  D          Read Distance to Destination (from the FMS)
  Shift+D    Read Distance + time to Top of Descent (once computed in flight,
             e.g. "145 miles to top of descent: 00:16:58")

Location:
  C          Read Nearest City

Trim:
  Shift+T    Toggle Trim Announcements On/Off

Tools:
  Ctrl+H     Activate/deactivate Hand Fly mode
  Ctrl+V     Activate/deactivate Visual Guidance mode. Does not require Hand
             Fly mode. Enables the same single-letter quick-access hotkeys
             Hand Fly mode does, plus F for target VS.
  Ctrl+T     Activate/deactivate Takeoff Assistance
  Ctrl+D     Read Distance to Selected Runway
  Ctrl+I     Read ILS Guidance System Report

Taxi Guidance (turn-by-turn taxi assistance with audio steering tone):
  Y          Taxi Status (current taxiway, next turn, distance to destination)
  Ctrl+Y     Repeat Current Taxi Instruction
  Alt+Y      Where Am I (announces current taxiway, gate, or runway at nearest airport)
  (Open the form with Input mode > Shift+Y)

Waypoint Tracking (use Fix Track window or EFB route viewer to add):
  1          Track Slot 1
  2          Track Slot 2
  3          Track Slot 3
  4          Track Slot 4
  5          Track Slot 5

Displays:
  Alt+E      Open E/WD (Engine/Warning Display) window — engine row, memos and
             active warnings. Read with the arrow keys; auto-refreshes, F5
             refreshes now, Escape closes.
  Alt+D      Describe outside scene (AI)
  Note: the PFD, ND, ISIS and System Display pages have NO hotkey window — read
        them from the Panels list (Ctrl+2): PFD / ND / Status under the Displays
        section, ISIS under Instrument, and the SD system pages from the ECAM
        Control Panel's "System Display Page" combo on the Pedestal.

Traffic and Weather:
  R          Announce Tracked TCAS Traffic
  Ctrl+R     Open TCAS Traffic Window
  Shift+R    Open Weather Radar Window (auto-detects ActiveSky and silently
             uses richer data — surface wind+gust, ceiling, QNH, turbulence,
             METAR-decoded precipitation — when running. Falls back to MSFS
             SimVars when AS isn't running. Visibility shown in both km and
             statute miles either way.)
  Alt+G      Announce nearest ground traffic (up to 3 aircraft on ground with
             direction and distance; auto-alerts also fire at 500/250/100 ft
             zone transitions while taxiing)

Displays and Windows:
  Ctrl+M     FlyByWire A380 Monitor Manager
  Ctrl+E     Toggle ECAM E/WD auto call-outs
  Ctrl+Shift+B  Airport Map & BTV (OANS) window
  Ctrl+Shift+C  Live Electronic Checklist (ECL) window
  Shift+E    Route viewer, procedure planner and airport lookup
  Shift+L    Location Info
  Shift+M    Show METAR Report (VATSIM, plus ActiveSky METAR if AS detected)
  Ctrl+Shift+T  Cold Temperature Altitude Correction (offline calculator: field elevation + temperature + published altitudes -> corrected altitudes)
  Shift+B    SimBrief Briefing
  Shift+C    Show Checklist (static text, cold-and-dark to shutdown)

Hand Fly Mode Hotkeys:
  A          Read MSL
  S          Read IAS
  D          Read Distance to Destination Runway
  H          Read Magnetic Heading
  P          Read Aircraft Pitch
  B          Read Aircraft Bank
  Q          Read AGL
  V          Read VS

Visual Guidance Mode Hotkeys (same single-letter set as Hand Fly mode, plus F):
  A          Read MSL
  S          Read IAS
  D          Read Distance to Destination Runway
  H          Read Magnetic Heading
  P          Read Aircraft Pitch
  B          Read Aircraft Bank
  Q          Read AGL
  V          Read VS
  F          Read Target VS

Note: both modes share the quick-access hotkeys. Activate Hand Fly mode,
Visual Guidance mode, or both — the single-letter keys H, V, Q, S, D, B,
P, A work in either. F is Visual-Guidance-specific (announces "Visual
guidance not active" if pressed when VG is off).

Ground Services:
  Ctrl+G     Read latest GSX tooltip (no need to open the Access GSX window)

Windows:
  Shift+M    Open MCDU / MFD window
  Shift+T    Open flyPad EFB window
  Ctrl+Shift+R  Open Radio Management Panel (RMP) window

Taxi Guidance:
  Shift+Y    Open Taxi Assist Form (enter ATC taxi clearance, start guidance)
  Y          Continue past current hold-short point
  Ctrl+Y     Stop Taxi Guidance
  Shift+X    Open Landing Exit Planner (pick a runway exit taxiway before
             touchdown; guidance auto-starts when you land — reuses the ILS
             destination runway if one is set; announces headwind/tailwind on
             runway selection so you can pick an exit that matches the rollout)

Ground Services:
  Alt+G      Open Access GSX window (accessible GSX Ground Services Pro menu)

General:
  Escape     Exit active hotkey mode

MCDU / MFD Window (open with [ then Shift+M):

Navigation:
  Up/Down        Move through lines
  Enter          Activate the line (send scratchpad to field; open a combo box)
  Backspace      CLR (scratchpad first, then a displayed info/error message, then field)
  Alt+C          CLR (same as Backspace)
  Delete         Clear the selected field
  Ctrl+Up        Previous page (PageUp also)
  Ctrl+Down      Next page (PageDown also; scrolls the F-PLN window)
  F5             Refresh
  Alt+S          Jump to the scratchpad input
  (type)         Type into the scratchpad, then Enter

MFD Page Buttons (Alt+letter):
  Alt+I  INIT     Alt+F  F-PLN    Alt+E  PERF     Alt+L  FUEL & LOAD
  Alt+R  RAD NAV  Alt+P  SEC FPL  Alt+D  DIR      Alt+A  ATC COM
  Alt+V  SURV     Alt+C  CLR      Alt+H  Refresh
  Alt+N  Toggle weight units (kilograms / pounds) for MSFSBA read-outs

Note: there are no UP/DOWN page buttons — page stepping is Ctrl+Up/Down or
PageUp/PageDown. The "go to MFD page" combo reaches ALL pages; arrow to one and
it navigates once your selection settles.

Radio Management Panel (RMP) Window (open with [ then Ctrl+Shift+R):
  The RMP screen is read-only text — read it with the arrow keys. It refreshes
  automatically and announces any RMP message (e.g. FREQ NOT VALID) as it appears.
  A side combo (Captain / First Officer / Overhead) switches which RMP you control;
  the screen, announcements and keys all follow it.
  Alt+V          VHF page
  Alt+T          Transponder (SQWK) page
  Alt+Home       Jump back to the screen (e.g. from a page button)
  (HF / TEL / NAV / MENU are not modelled by FBW — those do nothing)

  VHF page (left keys on Ctrl, right keys on Alt — same scheme as the MCDU):
  Ctrl+1/2/3     Select radio 1 / 2 / 3   (or F1/2/3 with alternate MCDU keys on)
  Alt+1/2/3      Swap that radio's active and standby   (or F7/8/9)
  (type)         Type the frequency (keyed live, no decimal — it auto-completes:
                 118500 or 1185 both give 118.500). Enter loads it into standby and
                 reads it back, e.g. "VHF standby 1, 118.500".
  Backspace      Delete a digit
  Alt+C          Fully clear a stuck entry

  Transponder (SQWK) page:
  (type)         Type the 4 squawk digits (0 to 7). It sets automatically on the 4th
                 digit (or press Enter) and reads back "Squawk 2222".
  Alt+I          Send IDENT

Live Electronic Checklist (ECL) Window (open with ] then Ctrl+Shift+C):
  The checklist is shown automatically when the window opens and hidden when it
  closes — no need to toggle it.
  Up/Down        Move the checklist cursor and read each line
  Enter          Open the highlighted checklist, or tick the highlighted item
  Backspace      Step back to the menu
  F5             Refresh
  (the Abnormal procedures button lists active ECAM procedures)

Airport Map / BTV (OANS) Window (open with ] then Ctrl+Shift+B):
  Accessible Brake-To-Vacate (BTV): pick your landing runway and the taxiway exit you
  want to leave at, and hear the predicted stopping distances. It reads the airport
  data directly and does NOT touch or zoom your navigation display. Read the controls
  with the arrow keys; Enter activates a button / opens a list; type into a field then
  Enter; Escape closes; F5 refreshes. A status line at the top says whether the full
  Navigraph maps are loaded, or only manual BTV is available. Three tabs:
   MAP AND BTV  Pick the BTV runway from the list + Arm runway, pick the exit + Arm
               exit, and read back: runway + landing distance available, exit +
               distance from threshold, the predicted dry / wet / live stop distances,
               runway-occupancy and turnaround times, and any "runway X ahead" caution.
               Without Navigraph maps, a Manual BTV section lets you type a stop
               distance (then select the landing runway in the FMS).
   AIRPORT     Load an airport's map: type the ICAO + Display airport, or use the
               Origin / Destination / Alternate buttons. Shows the loaded airport.
   STATUS      Whether the airport database is loaded + the AIRAC cycle.
  Full BTV (exit lists + predicted distances) needs a Navigraph subscription and being
  logged into Navigraph in the flyPad EFB (Shift+T). For taxi directions use Taxi
  Assist (Y / Shift+Y), which uses the navigation database directly.
  NOTE: the visual map controls (center map, crosses/flags, stands, de-icing, LDG
  SHIFT) are intentionally omitted — they have no audio meaning for a blind pilot.

Automatic Announcements:
  - Thrust-lever detents reached (Idle / Climb / Flex MCT / TOGA / Reverse)
  - Flight Mode Annunciator (FMA) — vertical, lateral and autothrust modes
  - ECAM warnings, cautions, memos, and the full E/WD failure stream (toggle Ctrl+E)
  - Master Warning / Master Caution
  - Altimeter setting, approach minimums, flaps and gear
  - Runway-overrun protection (ROW/ROP) and Brake-To-Vacate (BTV) rollout callouts
  - Overhead pushbutton FAULT lights and system warnings
  - Ground speed callouts while on the ground
  Mute any of these from the Monitor Manager (Ctrl+M).
